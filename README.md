## React + TypeScript + Webpack + Gulp Playground

### Setup

    npm install -g gulp
    npm install -g bower
    npm install -g typescript
    npm install -g tsd
    npm install -g webpack
    npm install -g webpack-dev-server

    npm install
    tsd install
    bower install

### Development - launch the development server

    gulp

### Package for production

    gulp webpack:build
